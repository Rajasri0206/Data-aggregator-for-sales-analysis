# Data Aggregator for Sales Analysis

A Python-based Sales Data Aggregator and Analysis System developed using Flask, Pandas, NumPy, and SQL Server integration with Pyodbc.  
The project provides REST API endpoints for managing and analyzing sales data efficiently.

---

## 🚀 Features

- REST API development using Flask
- Sales data storage and retrieval
- SQL Server database integration using Pyodbc
- Sales analytics and query processing
- Data analysis using Pandas and NumPy
- API testing and debugging using Postman
- Optimized SQL query execution
- Improved API reliability through bug tracking and fixes

---

## 🛠️ Tech Stack

- Python
- Flask
- Pandas
- NumPy
- Pyodbc
- SQL Server
- Postman
